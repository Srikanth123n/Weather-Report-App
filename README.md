# 🌦️ Weather Report App

A modern and responsive weather application built using **React.js** that provides real-time weather updates for any city using an external weather API.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature, humidity, and weather conditions
* 📦 Component-based architecture (React)
* 🎨 Clean and responsive UI
* ⚡ Fast data fetching using API integration

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS
* **API:** Weather API (e.g., OpenWeatherMap)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Srikanth123n/weather-app.git
```

2. Navigate to the project folder:

```
cd weather-app
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```

---

## 🔑 API Setup

* Get a free API key from a weather provider (e.g., OpenWeatherMap)
* Replace the API key in your project file where API calls are made
